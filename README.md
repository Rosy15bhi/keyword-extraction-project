This project was developed as part of the Data Mining and Text Analytics course.# Keyword Extraction Project

This project performs keyword extraction from text documents using Python and the RAKE algorithm.

## Project structure

- `data/` contains the input text file
- `outputs/` contains the extracted keywords output
- `src/` contains the Python source code

## Libraries used

- rake-nltk
- nltk

## How to run

Activate the virtual environment and run:

```bash
python src/keywords_rake.py
